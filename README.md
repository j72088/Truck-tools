# Trucker Tools Website

A lightweight React + Vite site for trucking calculators.

## Run locally

1. Install Node.js 18+.
2. Open this folder in a terminal.
3. Run:

```bash
npm install
npm run dev
```

## Build for deployment

```bash
npm install
npm run build
```

The production files will be created in `dist/`.

## Free hosting recommendation

### Vercel
- Import the project
- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

### Netlify
- Same build command and output directory
